## 🚀 Featured Projects

### [WTWR (What To Wear)](https://www.wtwr.computersforpeace.net/)
A smart clothing management app that helps users organize their wardrobe based on weather conditions. Features user authentication, CRUD operations for clothing items, and weather-based categorization.
- **Tech Stack:** React, Node.js, Express.js, MongoDB
- **Features:** User authentication, responsive design, RESTful API

### [ShinobiArchives](https://obito2912.github.io/shinobiarchives/)
An interactive encyclopedia for anime enthusiasts, currently focused on Naruto with plans to expand to multiple series. Built with modern React patterns and designed for scalability.
- **Tech Stack:** React, CSS3, JavaScript
- **Status:** In active development
- **Planned Features:** Multi-anime support, user authentication, community contributions
